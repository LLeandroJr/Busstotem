DESCRICAO DO PROJETO:

USUARIO PODE SABER SE EXISTE:
    linhas de onibus, que passam por esse terminal
    ou partem a partir dele para um cidade a qual deseja ir;

Informacoes que devem ser fornecidas pelo usuario/passageiro:
    digita o destino/cidade desejada;
    e o horario aproximado que deseja chegar ao destino.

O SISTEMA DEVE RETORNA PARA O USUARIO:
Sabe-se que as linhas de onibus tem as seguintes informacoes:
    numero de identificacao, que atribuido a linha;
O retorno do sistema para o usuario:
    numero da linha do onibus;
    a companhia responsavel pela linha do onibus;
    o horario de partida do onibus do terminal;
    e o horario de chegada do onibus ao destino.
